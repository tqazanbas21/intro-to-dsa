# Introduction to Data Structures and algorithms

This is a repository that containts Java based implementation of basic data structures &amp; algorithms and their tests written using testing framework Junit 4.

---

Directory structure of all the Data Structures or Algorithms were build using Maven Quickstart Archetype and the test phase of each build was handled by Maven Surefire Plugin. Maven can be installed via Arch Linux's package manager, to run the tests, type:
1. `cd DSA-name-here\`
2. `mvn compile`
3. `mvn test`
